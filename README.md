# sandbox-css-in-js-with-nextjs

## これはなに？ :tipping_hand_person:

css-in-js について学ぶために実際に手を動かすためのリポジトリです。
[styled-component](https://styled-components.com/) と [emotion](https://emotion.sh/docs/introduction) を試そうと思います。

## 起動方法 💻

以下のコマンドで実行してください。

```bash
npm ci
npm run dev
```
